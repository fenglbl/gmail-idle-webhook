# gmail-idle-webhook 项目信息

## 项目名称
gmail-idle-webhook

## 项目路径
/www/wwwroot/gmail-idle-webhook/

## 技术栈
- Node.js + ES Modules
- Express（HTTP API）
- imapflow（IMAP IDLE 连接 Gmail）
- mailparser（邮件解析）

## 核心功能
- Gmail IMAP IDLE 监听新邮件
- 新邮件到达时 Webhook 推送到配置的 URL
- REST API 管理配置和 webhooks
